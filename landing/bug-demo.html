Frequently Asked Questions

How does visual_check work?

perceptdot takes a screenshot of your URL using Cloudflare Browser Rendering, then analyzes it with Gemini 2.0 Flash to detect layout issues, overflow bugs, and visual regressions.

What counts as a visual check?

Each call to visual_check() counts as one check. Baseline comparisons count as two checks (before + after).

Can I use this in CI/CD?

Yes. Use the GitHub Action or call the API directly from your pipeline. Pro and Unlimited plans support CI/CD use cases.
