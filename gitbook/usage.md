## 기본 사용법

```

```

```
https://shop.example.com 레이아웃 문제 있어?
```

---

## 결과 예시

AI가 이렇게 알려줘요:

```
has_issues: true

문제 발견됨:
1. [HIGH] 상단 버튼이 컨테이너 밖으로 18px 튀어나옴
2. [MED]  모바일에서 텍스트가 잘려서 '...' 처리됨

전체 페이지 5개 구역 검사 완료 (2.3초)
```

---

## 자주 쓰는 표현

| 말하는 방법 | 뭐 하는 건지 |
|------------|------------|
| "확인해줘" | 전체 페이지 스캔 |
| "버그 있어?" | 문제만 골라서 알려줌 |
| "no\_cache=true로 다시 확인해줘" | 캐시 무시하고 새로 스캔 |

타일 1개당 크레딧 1개가 사용돼요.
