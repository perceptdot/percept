# ⚡ 1분 설치하기

터미널(검은 화면)을 열고 아래 명령어를 복사해서 붙여넣으세요.

{% hint style="info" %}
**터미널이 뭔지 모르겠다면?**\
Mac: Spotlight(⌘+Space) → "Terminal" 검색 → 엔터\
Windows: 시작 → "cmd" 검색 → 엔터
{% endhint %}

```bash
claude mcp add --transport http perceptdot https://mcp.perceptdot.com/mcp
```

엔터를 누르면 끝이에요! 진짜로요. 😄

---

## 잘 됐는지 확인하기

Claude Code에서 이렇게 말해보세요:

```
perceptdot으로 https://example.com 확인해줘
```

결과가 뜨면 성공이에요! 🎉
