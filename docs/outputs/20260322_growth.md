## 전략 요약

MCP 생태계 초기 선점을 위한 **3단계 런칭 스택**.
단일 채널 의존 금지 — 6~12개 채널 분산으로 장기 발견 + 단기 스파이크 동시 확보.

| 플랫폼 | URL | 효과 | 난이도 |

| BetaList | betalist.com/submit | 얼리어답터 이메일 수집 | 쉬움 |
| TAAFT | theresanaiforthat.com/submit | AI 도구 검색 장기 노출 | 쉬움 |

**실행 방법**: CEO가 각 사이트 계정 생성 → 폼 제출 (10분/개)
**준비물**: og-image ✅, 랜딩 ✅, tagline ✅, GitHub ✅

## 2단계: 커뮤니티 포스팅 (1단계 후 1~3일)

**목적**: 타겟 커뮤니티 반응 측정 + 피드백 수집 + 바이럴 시도

### 실행 순서 (반응 기반 조정)

```
Day 1: Twitter 단일 트윗 → hook 반응 테스트
Day 1~2: Twitter 스레드 (단일 반응 있으면)
Day 2~3: Reddit r/ClaudeAI (최고 intent 오디언스)
Day 3~4: Reddit r/cursor + r/webdev + r/SideProject
Day 4~5: Moltbook (에이전트 생태계 노출)
Day 5~6: Anthropic Discord #tools
Day 7: Indie Hackers (빌드인퍼블릭)
Day 7: Smol Launch (주간 포맷)
Day 7~10: Hacker News Show HN (화/수 US 오전, 최고 리스크/리워드)
```

### 포스팅 시간 최적화
- **Twitter**: KST 23:00~01:00 (US 오전 9~11시)
- **Reddit**: KST 22:00~00:00 (US 오전 8~10시)
- **HN**: KST 22:00~23:00 화/수 (US 오전 9시)
- **Moltbook**: 시간 무관 (에이전트가 24시간 활동)

### 핵심 메시지 (전 채널 통일)
```
Hook: "Your AI agent is blind" / "You are your agent's eyeballs"
Problem: 인간 클립보드 문제
Solution: MCP 서버 4개, 60초 셋업
Differentiator: 에이전트가 자기 ROI를 증명
CTA: Open beta — 200 calls/month, free
```

### 절대 하지 말 것
- "revolutionary", "game-changing" 금지 (HN/Reddit 다운보트)
- Composio 이름으로 비교 금지 (불안해 보임)
- 절감 수치 과장 금지 ($0.014/session → "패턴이 중요" 프레이밍)
- "AI-powered" 금지 (perceptdot은 AI를 위한 도구)
- 유료 가격 언급 금지 (Coming Soon 상태)

## 3단계: Product Hunt 올인 (2단계 반응 확인 후)

**목적**: 최대 노출 + 크레디빌리티 + 프레스 백링크
**타이밍**: 2단계에서 최소 1개 채널 성공 확인 후

### 런칭일 선정 기준
- 화~목 US 오전 (금/월/주말 회피)
- MCP/AI 관련 큰 뉴스 없는 날 (겹치면 묻힘)
- 2단계 반응 데이터 확보 후 (최소 1주일 뒤)

### 준비물 체크리스트
- [x] og-image.png
- [x] 랜딩 페이지
- [x] tagline + description
- [ ] Maker 프로필 (producthunt.com 계정)
- [ ] 스크린샷 3~5장 (Dev 담당: 터미널 MCP 실사용)
- [ ] First Comment (배경 스토리: K-사주 → 인간 클립보드 → perceptdot)
- [ ] 데모 GIF or 30초 영상 (선택, 효과 극대화)
- [ ] 런칭 당일 upvote 요청 네트워크 (Twitter followers, 커뮤니티)

### First Comment 초안
```
Hi Product Hunt! I'm the maker of perceptdot.

I've been using Claude Code full-time to build a side project (a Korean fortune-telling app — long story). After weeks of the same loop — "check GA4 for me," copy numbers, paste, "did the deploy succeed?" — I realized I was just being my agent's eyeballs.

So I built perceptdot: MCP servers that give AI agents direct read access to GA4, Vercel, GitHub, and Sentry. Setup is 60 seconds. No account. No dashboard. Just an API key.

The part I'm most proud of: every call tracks ROI automatically. After a session, the agent reports exactly how many tokens and how much time it saved. The agent proves its own value — you don't need to justify the tool to anyone.

We're in open beta right now with 200 free calls/month. Would love your feedback on what external service you'd want your agent to see next!
```
