### DOM false positive 수정 + 에이전트 운영 원칙 추가 ✅

**DOM 조상 overflow 클리핑 체크 개선 (커밋: 957d363)**
- 원인: DOM 감지 스크립트가 직계 부모만 체크 → ticker 내부 `span.flex`, `span.w-1.5` false positive
- 수정: `hasClippingAncestor()` 함수 추가 — 조상 전체를 순회하며 `overflow:hidden/scroll/auto/clip` 확인
- 효과: `overflow-hidden` 컨테이너 내부 요소(ticker, carousel 등) 더 이상 false positive 없음
- TypeScript 통과 + api.perceptdot.com 배포 완료 ✅

**에이전트 운영 원칙 추가 (CEO 지시)**

- `~/.claude/skills/po/SKILL.md`: 에이전트 모델 선택 섹션 강화, CEO 지시 명시

**이번엔 성공 확실해? — 검증 결과**
- DOM 탐지: 동일 URL 2회 연속 → 완전 동일 결과 ✅ (dom=7, 동일 셀렉터, 동일 픽셀값)
- False positive 수정: ticker overflow 이슈 해결 → 실제 버그만 탐지
- 남은 한계: CF Browser 동시 3회 빠른 호출 → 429 rate limit (코드 버그 아님, 인프라 한계)
