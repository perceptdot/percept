## PART 3: 돈이 되는 모델 TOP 3

### 1위: 컨시어지 서비스 (즉시 실행)

**무엇을 파나**: "Claude Code/Cursor에 MCP 서버 설정해드립니다"
**누가 사나**: MCP에 관심 있지만 설정 귀찮은 개발자, AI 도입 초기 팀
**가격**: Basic $49 / Pro $149 / Team $499
**왜 돈을 내나**: WordPress 설정 대행 시장 $10B+ — "나 대신 해줘"는 검증된 수요
**6개월 MRR**: 주 2건 x $49 = ~$400/mo. 프리미엄 포함 $500~800 현실적
**약점**: 확장성 없음. 그러나 현금흐름 + 고객 인사이트가 진짜 가치

### 2위: MCP 스타터 킷 (디지털 상품)

**무엇을 파나**: 사전 설정된 .mcp.json 템플릿 + 영상 가이드 + 추천 서버 목록
**가격**: $19 1회 결제 (Gumroad 이미 라이브)
**왜 돈을 내나**: "최적 MCP 조합이 뭔지 모르겠다" — 큐레이션 가치
**6개월 MRR**: 월 30건 = $570/mo. 콘텐츠 SEO 누적 시 자동화
**약점**: 생태계 변화 빠름, 콘텐츠 갱신 필요

### 3위: CFO 에이전트 (2~3개월 후)

**무엇을 파나**: "AI 도구 비용 감시 + 절감 방법 추천" MCP 서버
**누가 사나**: Cursor $200 + Claude API $100+ = 월 $300+ 쓰는 개발자
**가격**: 무료(리포트) / $9/mo (자동 알림 + 대안 추천)
**왜 돈을 내나**: AI 도구 비용 급증이 실제 고통. B2A2H 없이도 작동
**약점**: 비용 데이터 수집 API 제한적. MVP 2~3개월 필요
