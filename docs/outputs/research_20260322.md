# perceptdot 경쟁사 및 유사 서비스 조사 보고서
> 2026-03-22 | Research 에이전트 | 데스크탑 Claude

---

## 조사 범위
perceptdot의 4가지 핵심 차별점 기준으로 조사:
1. MCP 서버 플랫폼 (외부 서비스 연결)
2. ROI 자동 측정 (토큰/시간 절감)
3. 에이전트 자기추천 (B2A2H)
4. 에이전트가 사람에게 결제 권유

---

## 핵심 발견: "B2A2H"는 아직 없다

**"B2A2H" (Business to Agent to Human)** 모델은 웹에 어떤 검색 결과도 없음. perceptdot이 이 용어와 모델의 최초 정의자일 가능성이 높음. 에이전트가 스스로 도구를 추천하고 사람에게 결제를 권유하는 완전한 B2A2H 루프를 구현한 서비스는 현재 존재하지 않음.

### 1. perceptdot의 고유 포지션
**아무도 B2A2H 루프를 완성하지 못했음.** 토큰 절감을 측정하는 도구(Token Optimizer MCP, Helicone)는 있지만, 그 결과를 에이전트가 사람에게 보고하고 결제를 권유하는 서비스는 없음.

### 2. "ROI 보고"의 공백
옵저버빌리티 도구들(Helicone, Langfuse, LangSmith)은 모두 **개발자 대시보드**에 비용을 보여줌. 비개발자(CEO/사업주)에게 "이만큼 절감했습니다"라고 보고하는 도구는 없음. PwC가 이 니즈를 언급했지만 아직 제품화한 곳은 없음.

### 3. 직접 경쟁자 (기능 수준)
- GA4 MCP: 3개+ 존재 (Stape, harshfolio, surendranb) → **무료 오픈소스**
- Vercel: 네이티브 Agent Tools 이미 제공
- Sentry: 공식 MCP 서버 존재
- **차별점은 기능이 아니라 ROI 측정 + B2A2H 모델**

### 4. 수익화 인프라 활용 가능
Nevermined, MCPize, Moesif 등을 레이어 2 결제 인프라로 활용 가능

### 5. 시장 타이밍
- MCP 다운로드 85% MoM 성장
- 11,000+ MCP 서버 중 수익화 5% 미만
- 에이전트 커머스 시장 2030년 $1.7T 전망
- "에이전트가 결제를 권유" 모델은 아직 초기 — 선점 기회

### 6. 위협 요소
- Composio가 ROI 측정 기능 추가 시 가장 큰 위협
- Mintlify + Helicone 합병이 유사 영역 진입 가능
- GA4/Vercel/Sentry 공식 MCP 서버가 이미 무료로 존재 → 기능만으로는 차별화 불가

## 권장 사항

1. **"B2A2H"를 업계 용어로 확립** — 검색 결과 0건. 최초 정의자 위치 선점
2. **ROI 보고가 핵심 차별점** — 토큰 절감을 "비즈니스 언어"로 번역하는 것이 가치
3. **기존 GA4/Vercel/Sentry MCP 서버와 차별화 필수** — ROI 측정 없으면 기존 무료 서버와 경쟁 불가
4. **Token Optimizer MCP의 절감 보고 방식 참고** — 가장 유사한 "절감 보고" 구현체
5. **Nevermined/Moesif를 결제 인프라로 검토** — 바퀴 재발명 불필요
