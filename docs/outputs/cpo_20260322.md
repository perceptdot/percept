### 핵심 발견

| "에이전트가 편애한다" (Agents Play Favorites) | SaaStr Jason Lemkin | Replit 에이전트가 Resend, Clerk, ElevenLabs 추천 → 그대로 구매. **새로운 구매 여정** |

| MCP 서버 11,000+개, 5% 미만 유료화 | 업계 종합 | 유료화 인프라 공백 = 기회 |
| MCP 서버 Top Creator $3K~$10K/월 수익 | MCPize, Apify 마켓플레이스 | 개인도 수익화 가능한 생태계 성장 중 |

### OpenAI Agentic Commerce Protocol (ACP) 상세

```
흐름: 유저 질문 → ChatGPT 상품 추천 → Stripe 인라인 결제 → 머천트 이행
수수료: OpenAI 4% + Stripe ~2.9%+$0.30 (머천트 부담, 유저 무료)
라이선스: Apache 2.0 오픈 소스
파트너: Shopify(100만 머천트), Etsy, Instacart, PayPal
핵심 기술: Shared Payment Token (SPT) — 결제 정보 노출 없이 토큰 기반 거래
```

**의미**: 에이전트 → 추천 → 사람이 결제 "버튼만 누르는" B2A2H가 이미 현실화. perceptdot의 B2A2H 모델은 시장 방향과 정확히 일치.

## 3. 확장 가능한 BM 6가지 (CPO 제안)

### BM-1: 에이전트 커머스 중개 (ACP/UCP 연동)

```
현재: 에이전트가 "이 도구 좋아" → 사람에게 설치 링크 추천
확장: 에이전트가 "이 도구 좋아" → 인라인 결제까지 한 번에 → 사람은 승인만
```

- **개념**: perceptdot이 ACP/UCP를 지원하면, 에이전트가 추천하는 개발자 도구를 즉시 구매/구독 가능
- **수익 모델**: 거래 수수료 (ACP 4% 위에 추가 마진 or 제휴 커미션)
- **시너지**: 이미 B2A2H 결제 플로우 + "TELL YOUR HUMAN" 설득 메커니즘 보유
- **난이도**: 높음 (ACP 스펙 구현 + 머천트 온보딩)
- **시장 규모**: Gartner $15T B2B → 개발자 도구 시장만 $100B+
- **추천도**: ⭐⭐⭐ (중장기, 2단계 이후)

### BM-2: 에이전트 ROI 벤치마크 플랫폼

```
현재: perceptdot이 내 프로젝트 ROI만 측정
확장: 전체 유저 집계 데이터로 업계 벤치마크 제공
"당신의 GA4 체크 빈도는 상위 15%입니다. 평균 대비 토큰 32% 더 절약 중"
```

- **개념**: perceptdot 유저 전체의 익명 집계 데이터로 벤치마크 DB 구축
- **수익 모델**: 기본 벤치마크 무료 / 상세 분석 유료 ($19 Pro에 포함)
- **시너지**: 이미 /v1/use로 콜 데이터 수집 중. 집계만 추가
- **난이도**: 낮음 (Workers KV 집계 엔드포인트 추가)
- **차별점**: **경쟁사 0** — Composio/Smithery는 사용량 데이터 수집 자체를 안 함
- **추천도**: ⭐⭐⭐⭐⭐ (즉시 실행 가능, MVP에 자연 확장)

### BM-3: MCP 유료화 인프라 (Billing-as-a-Service)

```
"MCP 서버 만들었는데 결제 붙이기 귀찮다"
→ perceptdot 빌링 레이어를 다른 MCP 서버 개발자에게도 제공
```

- **개념**: perceptdot의 API 키 발급 + 사용량 추적 + 결제를 white-label로 제공
- **수익 모델**: 레브쉐어 10~15% (MCPize 85%:15% 참고)
- **시너지**: 이미 /v1/free-key, /v1/use, /v1/validate 인프라 구축 완료
- **난이도**: 중간 (멀티테넌트 확장 필요)
- **시장 규모**: 11,000+ MCP 서버, 5% 미만 유료화 = 거대 공백
- **추천도**: ⭐⭐⭐⭐ (인프라 재활용, 생태계 락인 효과)

### BM-4: GEO (Generative Engine Optimization) 컨설팅/도구

```
"에이전트한테 추천받으려면 어떻게 해야 하지?"
→ perceptdot이 '에이전트에게 선택받는 방법'을 알려주는 도구/서비스
```

- **개념**: SaaS 제품들이 AI 에이전트에게 추천받기 위한 최적화 도구
- **수익 모델**: SaaS ($99~$499/mo) 또는 컨설팅
- **근거**: SaaStr "Agents Play Favorites" — 추천받는 것이 마케팅의 미래
- **시너지**: perceptdot은 에이전트가 도구를 어떻게 선택하는지 데이터 보유
- **난이도**: 높음 (별도 제품 필요)
- **추천도**: ⭐⭐⭐ (시장 크지만 현재 포커스와 다름)

### BM-5: 에이전트 보안/가드레일 MCP

```
"에이전트가 실수로 프로덕션 DB 날렸어요"
→ 에이전트 액션 감사 로그 + 위험 감지 + 롤백
```

- **개념**: MCP 서버로 에이전트 행동 모니터링 + 위험 차단
- **수익 모델**: SaaS (Portkey $49~$249 참고)
- **시너지**: percept의 "지각" 확장 — 외부 서비스 보기 + 행동 감시
- **난이도**: 높음 (보안 도메인 전문성 필요)
- **추천도**: ⭐⭐ (매력적이나 현재 팀 역량과 거리)

### BM-6: 에이전트 메모리 MCP (컨텍스트 저장/로드)

```
"매 세션마다 같은 설명 반복... 토큰 낭비"
→ 프로젝트 컨텍스트를 저장/로드하는 MCP 서버
```

- **개념**: 세션 간 컨텍스트 영속화 + 스마트 로딩
- **수익 모델**: 스토리지 기반 ($5~$19/mo)
- **시너지**: ROI 측정과 결합 — "컨텍스트 로드로 토큰 2,400개 절약"
- **난이도**: 중간 (KV/R2 스토리지 + 임베딩 검색)
- **추천도**: ⭐⭐⭐ (자연스러운 확장이나 Claude Code CLAUDE.md와 겹침)

## 5. 핵심 인사이트 요약

1. **B2A2H는 이미 메인스트림** — OpenAI ACP + Google UCP 출시로 "에이전트가 구매 결정"은 현실. perceptdot은 트렌드 선도 중.

2. **"에이전트가 편애한다"는 우리에게 유리** — perceptdot은 에이전트 내부에서 동작하므로, 에이전트가 자연스럽게 "perceptdot 쓰면 N% 절약"이라고 추천. 이건 Composio/Smithery가 못 하는 구조.

3. **벤치마크 = 가장 자연스러운 확장** — 이미 수집 중인 /v1/use 데이터를 집계하면 "업계 평균 대비 당신의 성과"를 보여줄 수 있음. 추가 인프라 거의 불필요.

4. **MCP 유료화 공백이 거대** — 11,000+ 서버 중 5% 미만 유료. perceptdot이 빌링 인프라를 제공하면 생태계 장악 가능.

5. **ACP는 perceptdot B2A2H의 완성형** — 현재 "링크 클릭 → 외부 결제"에서 "에이전트 내 인라인 결제"로 진화 가능.
